# Vacations-project
Imaginary vacations project
A full-stack Vacations Management web application built with React, TypeScript, Node.js, Express, and MySQL.
The system includes authentication, role-based access (Admin/User), vacation CRUD operations, likes system, image uploads, reports, and AI-powered vacation recommendations.
Designed with a clean client-server architecture, REST API integration, Redux state management, and responsive modern UI.

Usage Instructions

1. Clone the repository.
2. Install all required dependencies in both the Backend and Frontend projects.
3. Configure the backend .env file with your MySQL database credentials and OpenAI API key.
4. Start the backend server.
5. Start the frontend application.
6. Open the application in the browser and login/register to use the system.
